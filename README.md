# CSS

Here we are studying CSS basic. In this course we learn every basic of CSS.

/////////////////////////////////////////////////////////////////////////////////

First of all we should know about what is CSS.
CSS is stands for cascading style sheet.
CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.
We do our styling web pages like how our web page look to attract viewer. So that our most of viewer stay there.
